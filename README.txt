- Descricao

	Este projeto, trata-se da implementacao de compilador para a linguagem Portugol simplificada. Onde a especificacao da linguagem encontra-se em https://github.com/victorfp/Compilador/tree/master/compilador/Doc.

- Execucao
	
	HÃ¡ duas formas de realizar os Testes:
	1) Coloque os arquivos de testes na pasta Teste com a extensao '.txt' e na pasta doc execute:
		- Windows
			execute o script compilador.bat;
			
		- Linux(testado no fedora 20)(desatualizado)
			no terminal execute:
				chmod +x .\compilador.sh
				.\compilador.sh
				
	2) Execute os script passando o arquivo de teste como paramentro
		- Windows
			compilador.bat Path_arquivo;
