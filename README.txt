- Descricao

	Este projeto, trata-se da implementacao de compilador para a linguagem Portugol simplificada. Onde a especificacao da linguagem encontra-se em https://github.com/victorfp/Compilador/Compilador/Doc.

- Execucao
	
	Para realizar os testes Lexicos, coloque os arquivos de testes na pasta Teste com a extensao '.txt' e na pasta doc execute o script compilador.bat, o qual ira realizar a analise lexicas em todos os arquivos.
