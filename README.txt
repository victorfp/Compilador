- Descricao

	Este projeto, trata-se da implementacao de compilador para a linguagem Portugol simplificada. Onde a especificacao da linguagem encontra-se em https://github.com/victorfp/Compilador/tree/master/compilador/Doc.

- Execucao
	
	Para realizar os testes Lexicos, coloque os arquivos de testes na pasta Teste com a extensao '.txt' e na pasta doc execute:
		- Windows
			execute o script compilador.bat;
			
		- Linux(testado no fedora 20)
			no terminal execute:
				chmod +x .\compilador.sh
				.\compilador.sh
		
	Com isso serao realizados os testes localizados na pasta Teste.
